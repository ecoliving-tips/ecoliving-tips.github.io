# Contributing to Swaram

Thank you for your interest in Swaram!

## Bug Reports and Feature Requests

We welcome bug reports and feature suggestions. Please [open an issue](https://github.com/ecoliving-tips/ecoliving-tips.github.io/issues) and include:

- A clear description of the bug or feature
- Steps to reproduce (for bugs)
- Screenshots if applicable
- Your browser and device info (for UI bugs)

## Pull Requests

This is a proprietary project. **Pull requests are not accepted** and will be closed without review.

## Code of Conduct

Be respectful and constructive in all interactions. Harassment, spam, or abusive behavior will result in being blocked.
