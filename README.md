# Swaram

**Free AI-powered chord finder + curated Malayalam Christian chord library.**

Upload any audio file and get instant chord detection — any genre, any language. Plus a hand-curated collection of Malayalam Christian devotional song chords for guitar and keyboard.

Live at: [ecoliving-tips.github.io](https://ecoliving-tips.github.io/)

## Features

- AI Chord Finder — upload audio, detect chords instantly
- Malayalam Christian devotional song chord library
- Guitar and keyboard chord diagrams (221 chords)
- Transpose, auto-scroll, offline support (PWA)
- Song request system
